HTML Tables

Table tags

Tag Name Description Preferable Use
<table> Table The wrapper element for all HTML tables. Have to begin with this if you want a table
<thead> Table Head The set of rows defining the column headers in a table. Preferable container for the column part of the table
<tbody> Table Body The set of rows containing actual table data. Preferable container for the values and data part of the table
<tr> Table Row The table row container. Must be used in order to have a table
<td> Table Data The table data container Uses for insert data and values
<tfoot> Table Foot The set of rows defining the footer in a table. Uses may vary, but it's semantics when used to display sums and synthesis information

Table Attributes

Attribute Name Description Preferable Use
colspan Column Span Defines how many columns a td element should span. Use it as a mixing cells in Excel would do
rowspan Row Span Defines how many rows a td element should span. Use it as a mixing cells in Excel would do
All other specific attributes are deprecated*
*Still supports all usage of HTML global attributes
